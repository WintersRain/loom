# Update Relationship Tracker

Update `relationship_tracker.md` with changes to MC's state, relationship progression, or key moments.

## When to Use

After scene events that change:
- MC's emotional state
- MC's physical state/location
- Relationship progression level
- Trust or resistance levels
- Key moments worth logging

## What to Update

### Current Status Section
- MC State level (if changed)
- Resistance Level
- Physical Comfort
- Emotional State

### Key Moments Log
Add new entries to the appropriate table:
- Resistance Moments
- Connection Moments
- Physical Comfort Progression

### MC's State Tracking
Update:
- Current Concerns
- Things MC Has Accepted
- Things MC Has Asked For

## Format

Use the existing format in the file. Add new entries, don't overwrite history.

## Rules

- Only update what actually changed
- Be specific about state transitions
- Include scene reference for new moments
- Don't inflate or deflate changes
